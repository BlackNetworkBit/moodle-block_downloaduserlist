Description :
This moodle block allows the Trainer/Manager/Admin to download a
csv-list of all Course-Members in the following Format :

username,firstname,lastname,Login method,First Login(course),Role

Installation :
This Plugin can only be added to an course.
It have 2 supported language files : de/en

Install-Steps:
1. Download the zip-file.
2. Go to the Adminestration->Site adminestration->Plugins->Install plugins
3. Select the downloaded file and follow the installation.
4. Go to your course and turn on block-editing
5. Select "Userlist-CSV" or in German "Teilnehmerliste-CSV"
6. Enjoy the Download funktion.
